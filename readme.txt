== Description ==

Questo plugin aggiunge un bottone di contatto rapido sempre visibile sul tuo sito WordPress. Facilita la comunicazione tra i visitatori del sito e il proprietario del sito, rendendo più semplice ed immediato il contatto o il link ad una risorsa/pagina importante.

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress.
